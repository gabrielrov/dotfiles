## ⚡️ Requirements

- [sesh](https://github.com/joshmedeski/sesh?tab=readme-ov-file#how-to-install) ([zoxide](https://github.com/ajeetdsouza/zoxide?tab=readme-ov-file#installation), [nvim](https://github.com/neovim/neovim/releases))

```sh
brew install sesh
```

- [tpm](https://github.com/tmux-plugins/tpm?tab=readme-ov-file#installation)

```sh
 git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

Type `<prefix> + I` to install plugins.
