## ⚡️ Requirements

- git [sudo]
- curl [sudo]
- stow [sudo]
- [homebrew](https://brew.sh/)

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

- FiraCode Nerd Font ([Download](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.3.0/FiraCode.zip)) - regular

## 📦 Installation

```sh
git clone git@github.com:gabrielrov/dotfiles.git ~/dotfiles
```

Use `stow` to link packages with `stow <package>`.
